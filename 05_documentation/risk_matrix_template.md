# Risk Matrix Template

Use this matrix to rate AI system risks based on likelihood and impact.

| Impact ↓ / Likelihood → | Low | Medium | High |
|--------------------------|------|--------|-------|
| **Low Impact**           | 1    | 2      | 3     |
| **Medium Impact**        | 2    | 4      | 6     |
| **High Impact**          | 3    | 6      | 9     |

Risk scores 7–9 = Critical  
Risk scores 4–6 = Moderate

# Contributing Guidelines

Thank you for your interest in contributing to the Ai_Ethics_Audit_Lab.

## How to Contribute
1. Fork the repository
2. Create a feature branch:
   git checkout -b feature/my-feature
3. Commit changes:
   git commit -m "Add new feature"
4. Push your branch:
   git push origin feature/my-feature
5. Open a Pull Request

## Coding Standards
- Use descriptive commit messages
- Follow PEP8 for Python
- Provide documentation for all new modules

## Ethical Standards
All contributions must:
- Respect privacy
- Avoid harm
- Include fairness considerations
- Support transparency and safety
