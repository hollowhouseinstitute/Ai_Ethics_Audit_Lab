# Security Policy

## Reporting a Vulnerability

If you discover a vulnerability:
1. Email: security@hollowhouse.org
2. Do NOT open a public GitHub issue
3. Provide:
   - Steps to reproduce
   - Potential impact
   - A suggested fix (if possible)

## Supported Versions
Security fixes are guaranteed for the *main* branch.

## Data Protection Rules
- Never commit sensitive data
- Use encryption for raw PII
- Run privacy checks before upload
